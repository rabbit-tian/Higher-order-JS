1. 如何切出一个psd图片
    - 选中图层
    - 右击 “复制图层”
    - 文档 => "新建"
    - 图像 => “裁切”
    - 保存：command + option + shift + s （这种保存方式 容量小）
    - 保存方式
        - PNG-24（有透明层）
        - jpeg（背景全白的）

2. 如何切jpeg的图
    - 使用切片工具，选择好要切的图片
    - 双击放大镜查看选择的情况
    - 文件 => 导出 => 保存为web格式
    - 注意： 选择 “**所有用户切片**” ，可以保存自己需要的图片
    
    
3. 如何把一张本地图片放在网上
    - 打开网站  https://sm.ms/
    - 将本地图片拖到对应位置
    - 点击upload，生成线上地址（ok了！）
